🍔 Yum-Yum 😋 – Food Delivery Website

Yum-Yum 😋 is a front-end food delivery web application built using HTML, CSS, and JavaScript.
The project simulates a real-world food ordering platform with user authentication, menu browsing, cart management, order history, and profile handling — all managed using LocalStorage.

🚀 Features

🔐 User Authentication

Register & Login system using LocalStorage

Session handling with currentUser

Logout functionality

🏠 Home Page

Attractive landing section with Swiper slider

Food categories & latest dishes

Dynamic welcome message for logged-in users

🍽 Menu & Search

Browse food items by category

Search food items

Quick view option for dishes

🛒 Cart & Orders

Add items to cart

Place orders

View complete Order History

Clear order history option

👤 User Profile

View profile details

Update profile information

Update delivery address

📞 Contact Page

Contact form with success animation popup

ℹ️ About Page

App introduction

3-step ordering process

Customer reviews slider

🎨 UI/UX Enhancements

Responsive design

Loader animations

Font Awesome icons

Swiper.js sliders

Download or clone the repository

Open login.html or register.html in your browser

Register a new account

Login to access the home page

Explore menu, place orders, view profile & order history

⚠️ No backend required – fully client-side project
